🧪 Multi-Subject CBT / UTME Mode

This page explains and launches CBT packages where several subjects are loaded at once. Teachers create the package from the CBT page by choosing how many subjects are being set, then uploading a CSV for each subject. Students receive one exam code and switch between subject tabs during the exam.

  1. Teacher opens CBT Manager.
  2. Enter title, class, term/session and duration.
  3. Under Multi-subject package, enter the number of subjects.
  4. Select each subject and upload/paste that subject CSV.
  5. Share the generated code. Students use the normal Take Exam page and see subject tabs.
